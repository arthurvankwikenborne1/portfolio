# Arthur's Dynamic Portfolio with AI Agent

Een interactieve porfoliowebsite met een geïntegreerde AI Agent waar bezoekers vragen kunnen stellen.

## 📦 Bestanden

- **index.html** - Hoofdpagina met alle secties (Hero, About, Skills, Projects, Experience, Chat)
- **styles.css** - Responsive design met moderne styling
- **profile-data.js** - Jouw profieldata (vaardigheden, projecten, ervaring)
- **script.js** - Chat-functionaliteit en dynamische content loading

## 🚀 Aan de slag

### 1. Open de Website

Zet `index.html` eenvoudig in je browser en de pagina laadt automatisch.

### 2. API Key Instellen

Bij het eerste bezoek wordt je gevraagd je API key in te voeren:

```
1. Verkrijg een API key bij je AI service provider
2. Plak deze in de prompt die verschijnt
```

Je API key wordt veilig in je browsergeheugen opgeslagen (localStorage).

### 3. Chat Starten

Scroll naar de "Ask Me Anything" sectie en stel vragen aan de AI Agent! De agent kent alles over:

- Jouw vaardigheden en technologieën
- Jouw projecten (inclusief het innovatieve AI + IaC project)
- Jouw ervaring en onderwijs
- Jouw carrièrevisie
- En meer!

## 🔧 API Key Beheren

**Key opnieuw instellen:**
Open de browser console en voer in:

```javascript
setAPIKey('je-nieuwe-api-key');
```

**Key verwijderen:**

```javascript
localStorage.removeItem('anthropic_api_key');
```

## 📝 Profielddata Aanpassen

Bewerk `profile-data.js` om je porfolio aan te passen:

- Voeg/verwijder projecten toe

- Wijzig je carrièrevisie
- Voeg ervaringen toe

De pagina laadt automatisch je wijzigingen!
